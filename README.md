# Simple Python ATM

A minimal, console-based ATM simulation written in Python.

---

## 🚀 Features

* **Balance Inquiry:** Check how much money is in the account.
* **Deposit:** Add money to the balance.
* **Withdrawal:** Take money out (prevents overdrawing).
* **Exit:** Safely close the application loop.

---

## 🛠️ How to Run

1. Make sure you have Python installed.
2. Open your terminal/command prompt.
3. Run the script:
```bash
python atm.py
